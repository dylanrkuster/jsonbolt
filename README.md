# ⚡ JSONBolt

**Fast JSON Formatter, Validator & Converter** — [jsonbolt.vercel.app](https://jsonbolt.vercel.app)

A modern, free, open-source JSON toolkit for developers. Format, validate, minify, convert, and compare JSON — all in your browser.

## Tools

| Tool | Description | Link |
|------|-------------|------|
| 🎨 Formatter | Beautify JSON with customizable indentation | [Open](https://jsonbolt.vercel.app/json-formatter) |
| ✅ Validator | Check JSON syntax with error positions | [Open](https://jsonbolt.vercel.app/json-validator) |
| 📦 Minifier | Compress JSON by removing whitespace | [Open](https://jsonbolt.vercel.app/json-minifier) |
| 🔤 Sort Keys | Alphabetically sort all keys | [Open](https://jsonbolt.vercel.app) |
| 🔄 JSON → YAML | Convert JSON to YAML | [Open](https://jsonbolt.vercel.app/json-to-yaml) |
| 🔄 YAML → JSON | Convert YAML to JSON | [Open](https://jsonbolt.vercel.app/yaml-to-json) |
| 📊 JSON → CSV | Convert JSON arrays to CSV | [Open](https://jsonbolt.vercel.app/json-to-csv) |
| 📄 JSON → XML | Convert JSON to XML | [Open](https://jsonbolt.vercel.app/json-to-xml) |
| 🔍 JSON Diff | Compare two JSON objects | [Open](https://jsonbolt.vercel.app/json-diff) |

## Features

- ⚡ Instant — results as you type
- 🔒 Private — 100% client-side, no server processing
- 🌙 Dark mode — easy on the eyes
- 📱 Responsive — works on mobile
- 🆓 Free — no signup, no limits, no ads

## Tech Stack

- Next.js 16 + TypeScript
- Tailwind CSS
- Vercel (hosting)
- js-yaml (YAML conversion)

## Development

```bash
npm install
npm run dev
```
